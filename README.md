# react-kod


react-kod is a free and light component Package

## Installation

react-kod requires react "^18.2.0", react-dome "^18.2.0" to run.

run this command:
```
npm i react-kod
```
add css to your project
```
import "react-kod/dist/index.css";
```
### Usage
import { DesiredComp } from "react-kod"
